# BrepKit WASM for OpenZCAD

This branch contains the browser package generated from
`esaueng/brepkit@3d2c4b186f84fb25e2fa798e1b9c98903a63e246` by the
`Build OpenZCAD WASM Release` workflow.

The generated JavaScript, TypeScript declarations, and WebAssembly module are
committed here so consumers can install the fork directly from an immutable Git
commit without resolving the package from npm.
